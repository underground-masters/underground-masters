# 재야의 고수들
다른 사람들과 재능을 교환하는 플랫폼, 재고입니다.

JavaFX와 JDBC를 기반으로 개인 간 재능 교환 게시글을 등록·조회·매칭할 수 있는 데스크톱 애플리케이션입니다.
사용자는 교환할 재능을 등록하고, 다른 사용자의 요청을 수락·거절하며 안전하게 연락처를 주고받을 수 있습니다.

## 주요 기능
- 자신의 재능 등록 및 수정
- 재능 교환 게시글 등록
- 등록된 게시글 조회
- 교환 매칭 리스트 확인
- 교환 수락 및 거절 처리
